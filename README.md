# NumPy Practice 🚀

This repository contains my hands-on practice of NumPy basics, covering array creation, indexing, operations, and real-world data manipulation.

---

## 📌 Program 1: NumPy Basics

### Concepts Covered:
- Creating arrays (1D & 2D)
- Understanding shape
- Indexing (row, column, element access)
- Mathematical operations
  - Addition
  - Element-wise multiplication
  - Matrix multiplication (dot product)
- Useful functions
  - Mean
  - Sum
  - Max / Min
- Reshaping arrays
- Special arrays
  - Zeros
  - Ones
  - Arange
  - Linspace

### Learning Outcome:
Built a strong foundation in NumPy basics and understood how arrays work internally.

## 📌 Program 2: Data Access & Manipulation

### Scenario:
Working with a dataset:
- Column 1 → Age
- Column 2 → Salary

### Concepts Covered:
- Accessing columns using slicing (`data[:,0]`)
- Accessing rows and elements
- Updating data (salary increase by 10%)
- Aggregations:
  - Average age
  - Maximum salary
- Partial slicing of data

### Learning Outcome:
Understood how to manipulate real-world data using NumPy arrays.

---

## 🧠 Key Concepts Learned

- Difference between:
  - `data[row][col]` vs `data[row, col]`
- Column access using slicing
- In-place vs assignment operations
- Importance of data types (`int` vs `float`)
